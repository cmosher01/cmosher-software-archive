Remote Control Reader

Windows 95 (or Windows NT 4.0) program to read data from the serial port COM1. You can download the source code in REMOTE.ZIP (14K). It requires MS Visual C++ V6.0 to build it. For now, this program just reads the remote signals, and creates a (large) log file containing signal timings and analysis. It is mostly a tool to help you study the remote control signals.

You need to build a device that reads the infrared signals from the remote and toggles the Carrier Detect signal on the serial port. Here's the schematic: 

           +-------------------+--------+
           |                   |        |
           |                 ----- C1   |
           |                 -----      |   +----------+
    +pin-5-+   D1             +|        +---+ GND      |
 DB9|pin-7-----|>|---+--/\/\---+------------+ Vcc   I1 |
    +pin-1-+         |   R1             +---+ Vout     |
           |         \                  |   +----------+
           |         /                  |
           |         \  R2              |
           |         /                  |
           |         |                  |
           +---------+------------------+
