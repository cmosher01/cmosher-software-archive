epanCount system

Java package overview
---------------------
com.surveysampling.emailpanel.counts.*:

[root]    = epanCount (GUI)
prototype = epanCountPrototype (GUI prototype)
update    = epanCountUpdate (data updater recurring job)
api       = epanCountLibrary (API)

api.geography = epanGeoParse (API for parsing geographic names)
api.criteria = deals with criteria for count requests (age, income, etc.)

api.list = deals with monitoring the list of requests for changes
api.list.monitor = generic monitoring of lists for changes

api.internal = classes used only internally in the API
api.internal.sql = classes dealing with database access

api.request = classes for count requests
