Spot Counts

Spot Counts is a program to run counts of the Spot database.

The current version is a prototype for the user interface. It does run actual counts of a snapshot of the Spot database. The snapshot is updated every Monday morning.

Spot Counts

After clicking the link, Java may need to download some things first, and it may ask you to accept some agreements or certificates. You should accept these. Then it may ask if you want to create program shortcuts, so you can easily run the program in the future without coming back to this web page.

After this preliminary setup, the Spot Counts program will start running. You will need to log in to the database using your username and password. After the program initializes, you can enter a new count and run it.

In order to see other people's counts (such as your team members' counts), you will want to go to File/Preferences and Ctrl-click on the people whose counts you want to see.

Please keep notes on any problems you discover, or comments or suggestions you have.
