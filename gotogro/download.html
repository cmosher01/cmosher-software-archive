Easiest way to download (Recommended)

1. Click on the "Download" link above.
2. Click Run this program from its current location, and click OK.
If a Security Warning comes up and asks you if you want to install GROv130.exe, click Yes.
This will automatically install GRO into the folder "C:\Program Files\GRO" and run it.

Expert way to download

If you want to install GRO into folder other than "C:\Program Files\GRO"
1. Click on the "Download" link above.
2. Click Save this program to disk, and click OK.
3. Use the Save As dialog to save GROv130.exe into whichever folder you want GRO installed.
4. Run GroV130.Exe from where you saved it.
