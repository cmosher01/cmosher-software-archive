Bulkemailer V1.1 Release Plan



PRE-ALPHA
---------
(SW as webuser or nsuser)
edit 3 .profiles to set and export SWLEVEL environment variable:
sagesse:~webuser/.profile SWLEVEL=alpha DONE
sagesse:~nsuser/.profile  SWLEVEL=beta
sparky:~webuser/.profile  SWLEVEL=release

copy /usr/ssi/deploy/app/bulkemailer/alpha/scripts/BulkEmailer.properties to:
sagesse:~webuser/bulkemailer/BulkEmailer.properties DONE
sagesse:~nsuser/bulkemailer/BulkEmailer.properties
sparky:~webuser/bulkemailer/BulkEmailer.properties
and edit as necessary (port and notification server)
(SW does all, except ISG edits sparky:webuser's properties file)



ALPHA (sagesse)
-----
(SW as deploy)
build bulkemailer 1-1 DONE
shutdown bulkmailer DONE
restore archive to alpha DONE

(SW as webuser) DONE
cd
rm submitBulkEmailerServer
cd bulkemailer
rm bem.ksh
mkdir mta
rm smtp*
rm job/job.xsd
mkdir history
mv log/Hist* history

(ISG as root)
cd /etc/init.d
rm emaildaemon_webuser
cp /usr/ssi/deploy/app/bulkemailer/alpha/scripts/bulkemaildaemon_webuser.ksh .
chmod 755 bulkemaildaemon_webuser.ksh
cd /etc/rc2.d
rm S99emaildaemon_webuser
ln -s ../init.d/bulkemaildaemon_webuser.ksh S99bulkemaildaemon_webuser
cd /etc/rc0.d
rm K99emaildaemon_webuser
ln -s ../init.d/bulkemaildaemon_webuser.ksh K99bulkemaildaemon_webuser
[also kill at other run levels (1, S) ???]

(SW as anybody) DONE
/etc/init.d/bulkemaildaemon_webuser.ksh start
set up MTAs: freebee 26, 27
