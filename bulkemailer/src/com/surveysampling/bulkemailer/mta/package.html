Provides a simple, thread-safe interface to a bank of email servers. To use this package, create one {@link com.surveysampling.bulkemailer.mta.MtaManager MtaManager} object and call its {@link com.surveysampling.bulkemailer.mta.MtaManager#addMta addMta} method for each email server to connect to. Then create (one or more) {@link com.surveysampling.bulkemailer.mta.MtaSession MtaSession} objects (passing the MtaManager to the constructor), and call its {@link com.surveysampling.bulkemailer.mta.MtaSession#send send} method to send an {@link com.surveysampling.email.Email Email}. When done sending messages, call {@link com.surveysampling.bulkemailer.mta.MtaSession#close close}.
